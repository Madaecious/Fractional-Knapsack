* To run the code simply open the file in an IDE and click run.

Note:
* The first line of the input.txt file is the capacity of the knapsack.
* The second line of the input.txt file is the names of the items in the knapsack.
* The third line of the input.txt file is the corresponding weights of the items in the knapsack.
* The fourth line of the input.txt file is the corresponding values of the items in the knapsack.
